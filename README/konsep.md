konsep dan rangkaian struktur utama :
1. tujuan website
    A. membangun personal branding profesional dan tetap asyik
    B. menunjukkan portofolio
        a. pamer codingan/repository di github
        b. pamer project gede
    C. karya blog/artikel (niche konten belum nemu nih)
    D. promosi jasa/produk saya
    E. cv online untuk melamar kerja
2. struktur utama website profile
    A. landing page/home
        a. foto diri
        b. nama lengkap
        c. tagline singkat
        d. CTA (call to action): contact me via instagram
    B. about me
        a. cerita singkat diri saya (2-3 paragraf)
        b. skill utama dan bidang keahlianmu
        c. nilai atau prinsip kerja
        d. foto diri casual
    C. skill and tool
        a. daftar skill teknikal dan soft skill yang relevan
        b. tools atau software yang di kuasai (beserta logo tool)
    D. portofolio/projects
        a. showcase 3-10 project terbaik (menggunakan grid)
        b. penjelasan singkat tiap project
            . nama project
            . deskripsi fungsi atau manfaat
            . tool/bahasa pemrograman yang diguanakan
            . link (github, live demo, video demo)
    E. experience/cv section
        a. riwayat pendidikan dan pekerjaan
        b. sertifikasi atau pelatihan relevan
        c. dengan menggunakan carousel
    F. testimoni dari klien
    G. blog/artikel
        a. tips belajar
        b. study case project
        c. pengalaman kerja
        d. topik personal branding
        e. dan lain-lain
    H. promosi produk digital
        a. produk affiliate 
        b. ebook 
        c. produk template apapun
    I. contact
        a. link ke media sosial
        b. link ke github
        c. email atau nomor wa
    J. footer
        a. copyright
        b. link cepat ke bagian penting website
        c. ucapan terima kasih
3. elemen visual dan UX
    A. pilih warna branding yang sesuai
    B. font profesional
    C. desain responsive (mobile friendly)
    D. tambahkan icon/SVG agar tampilan modern
    E. loading cepat dan clean
4. fitur teknis (upgrade lebih advanced)
    A. dark mode toggle
    B. animasi scroll reveal
    C. download CV dalam pdf
    D. live chat whatsapp
    E. form kontak terhubung ke email
    F. integrasi google analytics untuk tracking

catatan:
1. semakin kecil ukuran gambar, semakin cepat nantinya saat load
2. website statis dahulu sambil belajar js sekaligus upgrade dikit-dikit
